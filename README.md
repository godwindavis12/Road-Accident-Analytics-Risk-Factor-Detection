#  Road Accident Analysis Dashboard

##  Project Overview
This project presents an interactive Power BI dashboard analyzing road accident data to identify key risk patterns, severity distribution, collision types, and environmental factors influencing accidents.

The dashboard integrates dynamic DAX-based insights to provide contextual interpretation for each analytical section.

---

##  Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Slicers & Dynamic Measures

---

##  Key Analysis Areas
- Accident Severity Distribution
- Cause of Accident Patterns
- Light Conditions Impact
- Age Band Risk Analysis
- Area-wise Accident Concentration
- Collision Type Analysis
